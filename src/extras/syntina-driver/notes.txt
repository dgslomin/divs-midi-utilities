
Raspberry Pi setup:

	/boot/firmware/config.txt:

		dtparam=i2c_arm=on
		dtoverlay=i2c-gpio,bus=3,i2c_gpio_delay_us=1,i2c_gpio_sda=23,i2c_gpio_scl=24

Dependencies for Surge, Carla, and the Syntina driver:

	sudo apt install cmake g++ libgpiod-dev libxrandr-dev libxinerama-dev libxcursor-dev libfreetype-dev libasound2-dev libcurl4-openssl-dev libgtk-3-dev libwebkit2gtk-4.0-dev libjack-dev libjansson-dev
	(mkdir 3rdparty && cd 3rdparty && git clone https://github.com/yasir-shahzad/NAU7802.git)

Surge:

	cmake -Bbuild
	cmake --build build --config Release --target surge-staged-assets

Syntina driver:

	https://github.com/yasir-shahzad/NAU7802/blob/master/src/NAU7802.cpp
